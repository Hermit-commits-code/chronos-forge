# ⚔️ Chronos Forge

A high-performance, biomed-inspired time tracking engine built for architects who value precision.

## 🚀 The Tech Stack
- **Backend:** Go (Gin Framework) + GORM
- **Frontend:** Next.js 15 (React Compiler enabled) + Tailwind CSS
- **Database:** PostgreSQL
- **Security:** JWT (Coming Soon)

## 🛠 Features
- **Project Toggling:** Instant switching between tasks with sub-millisecond overhead.
- **Atomic Transactions:** Guaranteed data integrity using SQL transactions.
- **TDD-First:** Every core route is verified with automated Go tests.

## 🏁 Getting Started
1. **Backend:** `cd backend && go run main.go`
2. **Frontend:** `cd frontend && npm run dev`
3. **Database:** Ensure Postgres is running on Port 5432.
