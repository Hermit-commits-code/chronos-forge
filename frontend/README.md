# ⚔️ Chronos Forge

A high-performance, precision-engineered time tracking engine built for architects who value productivity as a craft.

## 🚀 The Tech Stack

* **Backend:** Go (Gin Framework) + GORM + PostgreSQL
* **Frontend:** Next.js 15 (App Router) + Tailwind CSS + Recharts
* **Security:** JWT (JSON Web Tokens) with Bcrypt Hashing
* **Architecture:** Modular Component Pattern (Atomic UI)

## ⚒️ Features

* **Forge Intensity:** Real-time 7-day area chart visualization of "heat" (productivity).
* **Atomic Toggling:** Instant switching between projects with sub-millisecond overhead.
* **Secure Ledger:** JWT-protected routes ensuring your time data is yours alone.
* **Manual Forging:** Ability to retrospectively add and scrap (delete) shifts.

## 🧰 Getting Started

1.  **Database:** Ensure PostgreSQL is running on Port 5432.
2.  **Backend:** ```bash
    cd backend && go run main.go
    ```
3.  **Frontend:** ```bash
    cd frontend && npm run dev
    ```
